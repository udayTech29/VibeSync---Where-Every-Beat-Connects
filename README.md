## 📌 Project Description

VibeSync is a modern music streaming UI inspired by Spotify. 

This project is created as a CSS Major Project to demonstrate responsive design and UI development.

## 🚀 Features

- Responsive design

- Music player UI

- Sidebar navigation

- Playlist section

## 🛠️ Tech Stack

- HTML

- CSS


💡 Inspiration

This project is inspired by popular music streaming platforms but built with a unique identity and branding.
